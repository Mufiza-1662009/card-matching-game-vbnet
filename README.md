# Card Matching Game (VB.NET)

## 📌 Project Description
This project is a card-based memory game developed using VB.NET. The objective of the game is to match pairs of cards containing identical shapes by flipping two cards at a time.

## 🧠 Core Logic & Implementation
- Implemented card-flipping functionality using event handling in Windows Forms  
- Each card stores a hidden shape value used for matching comparison  
- When two cards are selected, their values are compared to check for a match  
- Matching cards remain visible, while non-matching cards are flipped back  
- Game state is continuously updated until all pairs are matched  

## 🛠️ Technologies Used
- VB.NET  
- Windows Forms
